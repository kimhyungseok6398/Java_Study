## 1. 변수란 무엇인가?

* 자료형 변수명 = 값; = 값이 메모리 저장
* 같은 재료 -> 하나만 생성(상수)

## 5. 기본 자료형
* 숫자를 저장하는 자료형
* 정수형 : 소수점이 없는 수
* byte : 1바이트 -2^7 ~ 2^7-1 ( -126~125 ) 
* short : 2바이트 
* int : 4바이트 
* long : 8바이트 
 
 
## 문자형 
* char : 2바이트 
* 참고)   숫자 변수에 대입 숫자 그대로 대입 
* 문자를 변수에 대입할경우 '문자'
 
## 6. 실수형
